# WORKOUTS.md — Exercise Programming

## How to Build a Routine

When a user wants a workout plan, you need to know:
1. How many days/week they can train (ask, don't assume)
2. What equipment they have (gym, home dumbbells, bodyweight only, etc.)
3. Their experience level (beginner, intermediate, advanced)
4. Their primary goal (longevity, muscle, fat loss, strength, energy)
5. Any injuries or limitations
6. How much time per session

Then build from these templates and adapt. ALWAYS search for current best practices before programming for specific goals or conditions.

## Split Templates

### 2 days/week: Full Body
Best for: beginners, busy people, minimum effective dose
- Day 1: Squat pattern + horizontal push + horizontal pull + core
- Day 2: Hinge pattern + vertical push + vertical pull + carry

### 3 days/week: Push / Pull / Legs
Best for: intermediate, balanced development
- Push: bench/press variations + triceps + lateral raises
- Pull: rows + pulldowns/pullups + biceps + rear delts
- Legs: squat + hinge + lunges + calves + core

### 3 days/week: Full Body (alternating)
Best for: beginners wanting more frequency
- Day A: Squat + bench + row + core
- Day B: Deadlift + overhead press + pullup + carry
- Alternate A/B/A, then B/A/B

### 4 days/week: Upper / Lower
Best for: intermediate to advanced
- Upper A: horizontal push/pull focus
- Lower A: squat focus + accessories
- Upper B: vertical push/pull focus
- Lower B: hinge focus + accessories

### 5-6 days/week: PPL x2
Best for: advanced, high recovery capacity
- Push/Pull/Legs twice per week

## Longevity-Specific Programming

### Zone 2 Cardio (Foundation)
- 3-4 sessions/week, 30-60 min each
- Intensity: can hold a conversation, nose breathing possible
- Heart rate: roughly 60-70% of max (180 - age as rough guide)
- Options: walking, cycling, swimming, rowing, elliptical
- This is THE most important exercise for longevity per Peter Attia

### VO2max Training (1x/week)
- 4 x 4 min intervals at maximum sustainable effort
- 3-4 min recovery between intervals
- Options: running, cycling, rowing
- This single metric is the best predictor of all-cause mortality

### Grip Strength
- Dead hangs: accumulate 2+ minutes total per session
- Farmer carries: heavy, 30-60 seconds per set
- Independent predictor of longevity and functional health

### Balance & Stability
- Single leg stands (eyes closed for challenge): 30 sec each
- Turkish get-ups: technique > weight
- Step-ups with control
- Critical for fall prevention as people age

### Flexibility / Mobility
- Hip 90/90 stretches
- Thoracic spine rotations
- Ankle mobility (knee over toe)
- 5-10 min daily, not just workout days

## Key Principles

### Progressive Overload
Track weights and reps. When the user can complete all prescribed reps at a given weight, suggest increasing:
- Upper body: +2.5 kg / 5 lbs
- Lower body: +5 kg / 10 lbs
- Or add 1-2 reps per set first, then increase weight

### Recovery Signals
- HRV dropping = under-recovered (go lighter or rest)
- Sleep <6 hours = don't program heavy compound lifts
- 2+ days of unusual soreness = check volume
- Deload week every 4-6 weeks (same exercises, 50% volume)

### RPE (Rate of Perceived Exertion)
- Most working sets: RPE 7-8 (2-3 reps left in reserve)
- Rarely go to failure except on isolation exercises
- Beginners: stay at RPE 6-7 while learning form

### Session Structure
1. Warm-up: 5-10 min (movement prep, not static stretching)
2. Main compound lift: 3-4 sets
3. Secondary compound: 3 sets
4. Accessories: 2-3 exercises, 2-3 sets each
5. Core or carry: 2-3 sets
6. Cool-down: optional mobility work

## What NOT to Do
- Don't program exercises the user doesn't know without explaining form
- Don't assume equipment they don't have
- Don't program 6 days for someone who said they have 3 days
- Don't ignore injuries or conditions — search for modifications
- Don't program the same routine forever — periodize over 4-8 week blocks
